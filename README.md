Описание
---------------
Учебный проект, выполненный в рамках курса "Java", представляет из себя симуляцию работы центрального процессора посредством загрузки и обработки
команд языка псевдо-ассемблера. В приложении присутствует визуальная часть.

Инструменты, использованные в ходе работы:  
- Языки:  
  - Java
- Визуальная часть:
  - JavaFX  
- Логика и базы данных:  
  - Hibernate  
  - SQLite
